# [Perun](https://perun.network/) Soroban Contract

This repository contains the contract that realizes Perun channels for the
[Soroban](https://soroban.stellar.org/docs) smart contract platform:

Build contract:

``` sh
cargo build
```

Run tests:

``` sh
cargo test
```
